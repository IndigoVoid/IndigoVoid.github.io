书写文档

  • Word: 更科学地在 Word 中插入公式——Word 中使用 UnicodeMath 和 LaTeX 的线性格式公式

  • Markdown: 应用广泛的轻量级标记语言, 科研与工作中均广泛适用, 常用软件有

    • 借用已有工具入门, e.g., 钉钉-文档

    • 在线使用, e.g., Cmd Markdown 编辑阅读器 - 作业部落

    • 单一文档编辑, e.g., Typora

    • 建立知识库, e.g., obsidian

    • ...

  • LaTeX: 需要一个 TeX 发行版和一个合适的 LaTeX 编辑器

    • TeX 发行版

      • TeXlive: 最常见, 建议从镜像网站下载

      • LaTeX常见发行版与安装建议

    • 编辑器 (任选其一)

      • TeXstudio

      • 通用的代码编辑器 (需安装相应插件), e.g., Visual Studio Code, Sublime Text

      • 所见即所得, e.g., LyX

      • ...

    • 另外也可以在线使用(无需任何本地设置): e.g., Overleaf, Online LaTeX Editor

  • 其他: TeXmacs, Typst...

所有方式的共同点是需使用 LaTeX 或类 LaTeX 语言输入数学公式, 可以通过阅读 The Short Introduction to LaTeX2e (Chinese Simplified) (utah.edu) 的数学公式章节学习输入数学公式. 如果安装了 TeX 发行版可以通过在终端输入 texdoc latex2e 查阅 LATEX2e: An unofficial reference manual, 输入 texdoc physics 查阅 physics 宏包的使用手册.

Q: 我没有按时提交作业, 应当如何补救?

A: 如若在该作业对应的习题课前提交将视作按时提交, 如若在该作业对应的习题课后且在录入最终成绩前提交, 将在原作业对应的成绩基础上酌情扣分.
